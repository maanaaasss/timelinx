The `@webpacked-timeline/ui` package provides drop-in timeline components.

## DaVinciEditor

The main timeline editor component.

```tsx
import { DaVinciEditor } from '@webpacked-timeline/ui';
import '@webpacked-timeline/ui/styles/davinci';

function App() {
  return <DaVinciEditor />;
}
```

## Imports

| Import | Description |
|--------|-------------|
| `@webpacked-timeline/ui/styles/davinci` | Main stylesheet |
| `@webpacked-timeline/ui/styles/tokens` | CSS custom properties |
